## The Voltex Network

A multi-product AI ecosystem spanning research, autonomous agents, financial infrastructure, and cyber defense.

| Project | What it is | Status |
|---------|-----------|--------|
| [**FCRI**](https://github.com/Davidcarmelalex/fcri) | Fusion Civilization Research Institute — frontier science, NRLink, AZRAEL, MR NOTHING | 🔨 Active |
| [**VoltexBazar**](https://github.com/Davidcarmelalex/voltexbazar) | AI agent marketplace — deploy agents without being an infra engineer | 🔨 Active |
| [**Ayooni**](https://github.com/Davidcarmelalex/ayooni) | Cognitive AI engine — reasoning, planning, agent orchestration | 🔨 Active |
| [**AIKA**](https://github.com/Davidcarmelalex/aika) | Conversational AI — WhatsApp, Telegram, Voice | 🔨 Active |
| [**Automation**](https://github.com/Davidcarmelalex/automation) | Execution layer — Agent47, skills, workflows, Web3 ops | 🔨 Active |
| [**GrossphereOne**](https://github.com/Davidcarmelalex/grosphereone) | AI automation platform for businesses | 🔨 Active |
| [**AZRAEL**](https://github.com/Davidcarmelalex/fcri/tree/main/azrael) | Autonomous cyber defense — self-healing, sovereign, open | 🚧 Building |
| [**NRLink**](https://github.com/Davidcarmelalex/fcri/tree/main/apps/nrlink-api) | Cross-border liquidity orchestration | 🚧 Building |
| [**MR NOTHING**](https://github.com/Davidcarmelalex/MrNothing) | Android-native AI agent — for hustlers, dreamers, builders | 🚧 Building |

## System Architecture

```
Human Interaction
      │
      ▼
   AIKA  ──────────────────────── Telegram / WhatsApp / Voice
      │
      ▼
  Ayooni  ─────────────────────── Cognition / Planning / Routing
      │
      ▼
 Automation  ────────────────────  Agent47 / Skills / Web3
      │
      ▼
 GrossphereOne  ─────────────────  Business workflow layer
      │
 ─────┼──────────────────────────────────────────────────────
      │           FCRI Research Ecosystem
      ├── VoltexBazar    (AI agent marketplace)
      ├── NRLink         (cross-border settlement)
      ├── AZRAEL         (autonomous cyber defense)
      └── MR NOTHING     (Android-native AI agent)
```

## Stack

**Frontend** — Next.js 16 · React 19 · TypeScript · Tailwind CSS 4

**Backend** — Python 3.11+ · FastAPI · Node.js · Express · Prisma · PostgreSQL

**AI/Agents** — Ollama · Custom orchestration · Multi-agent systems · LLM routing

**Mobile** — Android · Kotlin · Jetpack Compose

**Infra** — Docker · Nginx · systemd · Datadog · Redis · Qdrant

**Web3** — EVM chains · USDT/USDC settlement
